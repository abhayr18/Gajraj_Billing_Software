# Gajraj Billing Software

A web-based **billing software** designed for small businesses to manage sales, generate bills, and maintain product records efficiently. Built using **Next.js**, **TypeScript**, and **CSS**, this software provides a clean and intuitive interface for easy billing management.

---

## Features

- Add, update, and manage products and items.
- Generate bills for customers quickly and accurately.
- Maintain records of all transactions.
- User-friendly interface with responsive design.
- Organized project structure for easy maintenance and scalability.

---

## Tech Stack

- **Frontend:** Next.js, TypeScript, JavaScript, CSS
- **Backend:** Node.js (via API routes in Next.js)
- **Database:** SQLite
- **Version Control:** Git & GitHub
